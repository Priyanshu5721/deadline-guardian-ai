# Deadline Guardian AI

## Overview
Deadline Guardian AI is an AI-powered productivity assistant that helps users analyze tasks, estimate effort, identify priority, and manage deadlines effectively.

## Features
- AI Task Analysis
- Priority Detection
- Difficulty Estimation
- Estimated Hours
- Urgency Analysis
- Suggested Action Plan
- Modern Responsive UI

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Gemini API

## Installation

```bash
npm install
npm run dev
```

## Future Scope
- User Authentication
- Task History
- Firebase Integration
- Calendar Sync
- Email Notifications
